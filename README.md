## Quick Summary

- Runs fully in-browser — no uploads.
- Batch compress, preview, and download (ZIP).
- Supports JPG / PNG / WEBP / SVG.

## Quick Start

```bash
git clone https://github.com/yourusername/optipix.git
cd optipix
npm install
npm run dev
# open the URL printed by Vite (usually http://localhost:5173)
```

## Environment

- Optional `.env` (ignored by git): `VITE_FEEDBACK_EMAIL`, `VITE_WEB3FORMS_KEY`.

## Deployment

- Deploy to Cloudflare Pages (recommended). Demo: `https://sieben.qzz.io`.
